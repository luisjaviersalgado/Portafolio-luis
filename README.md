# Portfolio Luis Javier - React Premium Interactivo

Portafolio profesional en React + TypeScript con diseño premium, terminal interactiva, cambio de idioma ES/EN, 5+ proyectos y backend opcional en Kotlin + Spring Boot.

## Correr frontend

```powershell
npm install --legacy-peer-deps
npm run dev
```

Abrir:

```txt
http://localhost:5173/
```

## Cambiar foto

Reemplaza este archivo:

```txt
public/assets/profile.jpg
```

## Backend opcional

El backend está en:

```txt
portfolio-backend-api
```

Incluye endpoints para perfil, proyectos, tecnologías y contacto.
