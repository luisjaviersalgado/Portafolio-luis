# Portfolio Backend API

Backend opcional en Kotlin + Spring Boot para demostrar experiencia backend dentro del portafolio.

## Ejecutar

```powershell
cd portfolio-backend-api
./gradlew bootRun
```

En Windows, si agregas Gradle Wrapper:

```powershell
gradle wrapper
.\gradlew bootRun
```

## Endpoints

```txt
GET http://localhost:8080/api/profile
GET http://localhost:8080/api/projects
GET http://localhost:8080/api/technologies
GET http://localhost:8080/api/contact
```
